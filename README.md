# Dwarf - dotfiles

@tayutayura's dotfiles
## Toolset
- Neovim
  - dein
- Zsh
  - zplug
- asdf
  - Ruby
  - Python
  - Go
  - Nodejs
  - Erlang
  - Elixir
- GNU GLOBAL

## Install
```
git clone git@github.com:tayutayura/dwarf.git ~/.dwarf  
sh ~/.dwarf/dwarf create
```

## Usage
- dwarf link  
  link dotfiles.
- dwarf initialize  
  install toolset.
- dwarf create  
  link & initialize.
